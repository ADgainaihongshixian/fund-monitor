## 为 Badge 组件添加 padding 和 border-radius 样式

**当前代码**（第 55 行）：
```tsx
<Badge variant="standard" color="info" sx={{ 
  fontSize: '0.75rem', 
  border: '1px solid', 
  borderColor: 'info.main', 
  backgroundColor: 'background.paper', 
  color: 'info.main', 
  '& .MuiBadge-badge': { 
    border: '1px solid', 
    borderColor: 'info.main', 
    backgroundColor: 'background.paper', 
    color: 'info.main' 
  } 
}}>
```

**修改方案**：
在 `'& .MuiBadge-badge'` 对象中添加两个样式属性：
- `padding: '0 4px'` - 左右内边距 4px，上下 0
- `borderRadius: '4px'` - 圆角 4px

**修改后的代码**：
```tsx
<Badge variant="standard" color="info" sx={{ 
  fontSize: '0.75rem', 
  border: '1px solid', 
  borderColor: 'info.main', 
  backgroundColor: 'background.paper', 
  color: 'info.main', 
  '& .MuiBadge-badge': { 
    border: '1px solid', 
    borderColor: 'info.main', 
    backgroundColor: 'background.paper', 
    color: 'info.main',
    padding: '0 4px',
    borderRadius: '4px'
  } 
}}>
```

**预期效果**：
- Badge 标签将具有更紧凑的内边距（左右 4px）
- 圆角为 4px，使外观更柔和
- 与现有的边框和颜色样式保持一致
