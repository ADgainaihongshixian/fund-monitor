# 基金实时估值监控项目图标和样式优化计划

## 一、图标资源优化

### 1. MUI图标组件集成
- 导入MUI图标库：`@mui/icons-material`
- 将所有内联SVG图标替换为对应的MUI图标组件
- 确保图标风格统一，符合MUI设计规范

### 2. 图标映射方案
- **导航栏图标**：
  - 基金图标 → `AccountBalance`
  - 刷新图标 → `Refresh`
  - 添加图标 → `Add`
  - 设置图标 → `Settings`
- **状态栏图标**：
  - 时间图标 → `AccessTime`
  - 自动刷新状态点 → 使用MUI `Badge`组件
- **错误提示图标**：
  - 错误图标 → `ErrorOutline`
- **模态框图标**：
  - 关闭图标 → `Close`
  - 成功图标 → `CheckCircle`
  - 警告图标 → `Warning`

## 二、图标尺寸调整

### 1. 统一图标尺寸规范
- **导航栏图标**：20px (MUI `fontSize="small"`)
- **按钮内图标**：16px (MUI `fontSize="small"`)
- **状态栏图标**：14px (MUI `fontSize="small"`)
- **模态框图标**：24px (MUI `fontSize="medium"`)
- **错误提示图标**：20px (MUI `fontSize="small"`)

### 2. 图标与文字比例优化
- 确保图标与相邻文字大小比例协调
- 调整图标与文字之间的间距，保持视觉平衡

## 三、样式一致性检查与优化

### 1. 颜色系统统一
- 确保所有颜色使用MUI主题或Tailwind配置的颜色变量
- 统一按钮、卡片、文本等元素的颜色方案

### 2. 间距系统优化
- 使用MUI或Tailwind的间距系统，确保一致的间距规范
- 优化组件之间的间距，提升界面呼吸感

### 3. 字体系统统一
- 确保字体大小、字重、行高的一致性
- 建立清晰的字体层级，提升信息可读性

### 4. 组件样式标准化
- 统一按钮、卡片、输入框等组件的样式
- 确保组件状态（默认、悬停、点击、禁用）的视觉一致性
