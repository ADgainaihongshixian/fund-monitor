## 添加缺失的背景色工具类

**问题**：项目中使用了 `bg-success`、`bg-info` 和 `bg-danger`，但这些类没有被定义

**修复方案**：

在 `src/index.css` 文件的 `@layer components` 部分添加以下背景色工具类：

```css
.bg-success {
  background-color: var(--success-color);
}

.bg-info {
  background-color: var(--info-color);
}

.bg-danger {
  background-color: var(--danger-color);
}
```

**设计考虑**：
- 使用与现有 `.text-*` 工具类相同的 CSS 变量
- 保持与 MUI 主题颜色的一致性
- 支持项目中所有语义化颜色（success、info、danger、warning）
- 颜色值已在 `:root` 中定义，确保主题统一

**预期结果**：
- 状态指示器圆点将正确显示颜色
- 所有使用 `bg-success`、`bg-info`、`bg-danger` 的地方都能正常工作
