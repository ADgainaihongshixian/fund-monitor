# FundList组件图标优化方案

## 问题分析

根据用户要求，需要将FundList组件中的SVG图标替换为MUI图标库中的图标组件，以确保图标风格统一且便于维护。

## 优化方案

### 1. 图标替换计划

| 当前SVG图标 | 功能描述 | 替换为MUI图标 |
|------------|---------|-------------|
| 基金图标 | 表示基金相关功能 | AccountBalance |
| 勾号图标 | 表示输入基金代码 | CheckCircle |
| 搜索图标 | 表示搜索基金名称 | Search |

### 2. 实施步骤

1. **导入MUI图标组件**：在FundList.tsx文件顶部导入所需的MUI图标组件
2. **替换SVG图标**：将文件中的SVG图标标签替换为相应的MUI图标组件
3. **调整图标尺寸**：确保图标尺寸与周围元素协调一致
4. **验证修改效果**：确保修改后的图标显示正常且风格统一

### 3. 具体修改

- **导入语句**：添加 `import { AccountBalance, CheckCircle, Search } from '@mui/icons-material';`
- **基金图标**：替换为 `<AccountBalance className="h-16 w-16 text-primary mx-auto" />`
- **勾号图标**：替换为 `<CheckCircle className="h-5 w-5 text-info" />`
- **搜索图标**：替换为 `<Search className="h-5 w-5 text-info" />`

### 4. 预期效果

- 所有SVG图标被替换为MUI图标组件
- 图标风格统一，与其他组件保持一致
- 图标尺寸与周围元素协调
- 代码更加简洁，便于维护
