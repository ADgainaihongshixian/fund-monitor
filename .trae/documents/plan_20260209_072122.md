## 为 IconButton 组件添加 margin-left 并修改 padding

**当前代码**（第 72-82 行）：
```tsx
sx={{
  color: 'info.main',
  '&:hover': {
    color: 'error.main',
    backgroundColor: 'action.hover',
    transform: 'scale(1.1)',
  },
  transition: 'all 0.3s ease',
  p: 0.75,
  borderRadius: '50%'
}}
```

**修改方案**：
1. 添加 `ml: '4px'` - 左外边距 4px
2. 将 `p: 0.75` 改为 `p: '3px'` - 内边距改为 3px

**修改后的代码**：
```tsx
sx={{
  color: 'info.main',
  '&:hover': {
    color: 'error.main',
    backgroundColor: 'action.hover',
    transform: 'scale(1.1)',
  },
  transition: 'all 0.3s ease',
  ml: '4px',
  p: '3px',
  borderRadius: '50%'
}}
```

**预期效果**：
- IconButton 与左侧内容保持 4px 的间距
- 内边距从 0.75（12px）减小到 3px，使按钮更紧凑
- 保持现有的悬停效果和过渡动画
