## 路径别名配置方案

项目使用 Vite + TypeScript，需要同时配置两个文件：

### 1. 修改 `vite.config.ts`
- 添加 `resolve.alias` 配置
- 使用 `path` 模块解析绝对路径

### 2. 修改 `tsconfig.json`
- 添加 `baseUrl: "."`
- 添加 `paths` 映射配置

### 配置的别名
| 别名 | 映射路径 |
|------|----------|
| `@/*` | `src/*` |
| `@components/*` | `src/components/*` |
| `@hooks/*` | `src/hooks/*` |
| `@services/*` | `src/services/*` |
| `@stores/*` | `src/stores/*` |
| `@types/*` | `src/types/*` |
| `@utils/*` | `src/utils/*` |

### 使用示例
```tsx
// 之前
import { FundData } from '../../types';
import { formatNumber } from '../../utils/formatter';

// 之后
import { FundData } from '@/types';
import { formatNumber } from '@/utils/formatter';
```
